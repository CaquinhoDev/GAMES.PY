# GAMES.PY
 Jogos em python bem divertidos!

## COMO RODAR O BOT?

Siga essas instruções para poder rodar o bot sem problemas:

1. **Instalar as dependências do projeto:**

    ```bash
    pip install pygame
    ```

2. **Jogo da velha:**

    ```bash
    python velha.py
    ```

3. **Jogo ping pong:**

 ```bash
   python pong.py
 ```
